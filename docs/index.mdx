# Stylable Components

## Adding to your project

First you need to add a dependency; to do so, run:

<Tabs groupId="node-package-manager">
    <TabItem value="npm" label="npm">
        <CodeBlock language="bash">
            npm i --save stylable-components
        </CodeBlock>
    </TabItem>
    <TabItem value="yarn" label="yarn">
        <CodeBlock language="bash">
            yarn add stylable-components
        </CodeBlock>
    </TabItem>
</Tabs>
